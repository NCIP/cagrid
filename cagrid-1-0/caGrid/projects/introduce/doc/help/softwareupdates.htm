Introduce Graphical Development Environment

Software Updates

Introduce Software Update

Introduce is capable of downloading and installing new extensions, upgrades to older extensions, and newer versions of itself. In the GDE there is a Help menu. In this menu there is a Check for Updates button. This button will take the user to a wizard which will walk them through looking for any software updates or new packages which they may want to download and install. The user can also put in a different software update site URL to point to a custom site containing Introduce extensions.
